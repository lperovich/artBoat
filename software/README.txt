CODE OUTLINE

BOATCOLORCHANGEDESIGN2
**Code for the on-shore control board to select and send colors to the boat
**Rotary potentiometer controls the fade speed of the lights (not included in design one)
**Sliders control red, green, blue components of the buttons
**Any single button press sends the color to the remote boat
**Holding down a button in the array saves that color for later use

COLORBOATRXDESIGN2
**Code for on-boat receiving of information from shore and light control
**Expects colors and fade times for transitions between colors (goes w/designs that include rotary potentiometer)
